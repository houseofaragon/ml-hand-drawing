# ML Hand Drawing
This project users TensorFlow Hand Pose Detection model
along with the browser camera as input to enable a user to draw onto a canvas on the screen.

Technologies used:
- TensorflowJS
- React
- Typescript
- HTML5 Canvas

https://blog.tensorflow.org/2021/11/3D-handpose.html
